> An old school FPS game in C++ inspired by `Wolfstein 3D`

## Install

```sh
git clone https://github.com/siddhantrao23/sadenstein.git
sudo apt install -y libsdl2-dev git
cd sadenstein
mkdir build
cd build
cmake ..
make
```

## Usage

```sh
./sadenstein
```
